# Screenshots

Capturas para el README principal. Cuando `rootea.es` esté en
producción y verde, sustituye los placeholders de aquí por capturas
reales tomadas en navegador (preferentemente Safari/Chrome a
1440×900 con zoom 100%).

Convenciones:

- `home.png` — pantalla inicial con las tres tarjetas de stats.
- `all.png` — vista de la tabla maestra `/all` con scroll inicial.
- `machine.png` — página de una máquina (recomendado: `lame` o
  `forest`, son las más completas).

Una vez subidas, el README las enlaza vía:

```markdown
![Home](docs/screenshots/home.png)
```

Tamaño recomendado: ~1200px de ancho, formato PNG optimizado.
Comprime con `pngquant` o `imagemin` antes de commitear (peso
objetivo: < 200 KB cada una).
